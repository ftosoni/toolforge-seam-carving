# Contributing to Toolforge Seam Carving Utility

Thank you for your interest in contributing! We welcome contributions to improve this content-aware image resizing utility.

## 🛠️ Development Setup

To set up a local development environment, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ftosoni/toolforge-seam-carving.git
   cd toolforge-seam-carving
   ```

2. **Serve the files**:
   Since the app is pure HTML/CSS/JS, you don't need any complex installation. Simply start a local server, e.g.:
   ```bash
   python -m http.server 8000
   ```

3. Open `http://localhost:8000` in your web browser.
